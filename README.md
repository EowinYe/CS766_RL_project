# Play Games! Reinforcement Learning in Computer Vision
Dongqiangzi Ye, Haozhen Wu, Jennifer Cao

### Dependencies

+ Python 3.3+
+ TensorFlow >= 1.8.0
+ OpenAI Gym
+ StableBaselines

### Usage

If you want to run dqn, simply modify some parameters in the dqn.py and Run:

	python dpn.py

The same as other algorithms.
